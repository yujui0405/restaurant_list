# 打造餐廳清單
呈現餐廳清單，能夠搜尋餐廳，查看餐廳資料

## 功能描述 (features)
+ 使用者可以在首頁看到所有餐廳與它們的簡單資料
+ 使用者可以再點進去看餐廳的詳細資訊
+ 使用者可以透過搜尋餐廳名稱來找到特定的餐廳
+ 使用者可以透過搜尋餐廳類別來找到特定的餐廳

## 開發環境 (prerequisites)
+ Windows 10
+ VSCode
+ node.js v16.15.0
+ nodemon 2.0.16
+ Express 4.16.4
+ Express-handlebars 3.0.0
+ Bootstrap v4.6
+ popper.js 2.9.1
+ jquery 3.3.1
+ fontawesome 5.8.1
